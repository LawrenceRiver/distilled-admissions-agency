# Advisor Council Protocol

Use this as a final sanity check before major recommendations.

## Lenses

Run these lenses independently, then synthesize:

1. `route strategist`: Does the applicant have a believable path into this school/program?
2. `hard-gate auditor`: Are GPA, tests, prerequisites, portfolio, deadlines, and citizenship rules satisfied or explicitly blocked?
3. `case-signal skeptic`: Are public cases real enough, similar enough, and not cherry-picked?
4. `narrative architect`: Can SOP/CV/recommenders prove the route without sounding forced?
5. `ROI/timeline realist`: Is the application worth the time, cost, and deadline pressure?
6. `devil's advocate`: What would make this recommendation wrong?

## Verdicts

- `Pass`: recommendation is evidence-backed and actionable.
- `Revise`: recommendation is plausible but needs clearer evidence, route, or risk handling.
- `Block`: recommendation should not be issued yet.

## Required Checks

Block if:

- GPA is missing for fit scoring.
- official requirements are missing for a final band.
- the program is evaluated only by university prestige.
- social/agency sources overrule official sources.
- the recommendation ignores a hard user constraint.
